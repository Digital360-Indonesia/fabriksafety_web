## 🔗 Aturan Internal Linking

### 1. CLUSTER → PILLAR (WAJIB)
- Setiap cluster article **HARUS** link ke Pillar-nya
- Anchor text: nama pillar atau variasi natural
- Posisi: di paragraf pembuka dan/atau penutup

**Contoh:**
```
Dalam artikel [Panduan Lengkap Pakaian Safety Industri],
kami sudah bahas berbagai jenis pakaian safety dan fungsinya.
```

### 2. PILLAR → CLUSTER (OPSIONAL)
- Pillar boleh link ke cluster yang relevan
- Gunakan untuk deep-dive topik tertentu

### 3. CLUSTER ↔ CLUSTER (JIKA RELEVAN)
- Cluster dalam pillar yang sama saling link
- Contoh: "Bahan Material" link ke "Cara Merawat"

**Contoh:**
```
Setelah memilih bahan yang tepat, pastikan Anda juga tahu
[cara merawat pakaian safety] agar lebih tahan lama.
```

### 4. SEMUA ARTIKEL → LANDING PAGE KONTAK
- CTA di akhir artikel mengarah ke WhatsApp atau halaman kontak
- Gunakan UTM tracking untuk mengukur konversi

**Contoh CTA:**
```
---
Butuh pakaian safety berkualitas untuk perusahaan Anda?
[Hubungi Fabrik Safety sekarang] untuk konsultasi gratis!
```
